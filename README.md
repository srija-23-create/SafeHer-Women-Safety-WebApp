# SafeHer

## Overview
SafeHer is a team-developed Women Safety Web Application that provides emergency assistance through SOS alerts, emergency contact management, and live location sharing.

## Features
- User Authentication
- SOS Alert System
- Emergency Contact Management
- Live Location Sharing
- SOS History

## Tech Stack

- Flask
- React
- SQL
- HTML

- JavaScript
